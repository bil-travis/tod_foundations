To insert an image use the img tag, and then refer to either a local image as done below.

Or you can link to an absolute image. Besides the src attribute (used to include the image), there is also an alt attribute which is used to describe the image when it can not load.

Go ahead and resize the image using the height and width attributes. The size will be specified in pixels.
